============
Examples
============

We provide several examples so users can get a more practical grasp of how to use :code:`redback`.

Jupyter Notebook Tutorials
===========================

We have written comprehensive tutorial-style Jupyter notebooks demonstrating the functionality of :code:`redback`:

- `Main Redback Tutorial <https://github.com/nikhil-sarin/redback/blob/master/examples/RedbackTutorial.ipynb>`_ - Comprehensive introduction to redback
- `Gaussian Process Tutorial <https://github.com/nikhil-sarin/redback/blob/master/examples/GP_tutorial_with_redback.ipynb>`_ - Using Gaussian processes with redback
- `Redback Paper Plots <https://github.com/nikhil-sarin/redback/blob/master/examples/RedbackPaperPlots.ipynb>`_ - Reproducing plots from the redback paper
- `Target of Opportunity Simulations <https://github.com/nikhil-sarin/redback/blob/master/examples/ToOSimulations_with_redback.ipynb>`_ - Simulating ToO observations
- `Estimating Blackbody Properties <https://github.com/nikhil-sarin/redback/blob/master/examples/estimating_blackbody_properties_with_redback.ipynb>`_ - Blackbody parameter estimation
- `Estimating Bolometric Luminosity <https://github.com/nikhil-sarin/redback/blob/master/examples/estimating_lbols.ipynb>`_ - Calculating luminosities
- `Handling Non-detections <https://github.com/nikhil-sarin/redback/blob/master/examples/handling_non_detections.ipynb>`_ - Working with upper limits
- `Analyzing Spectra <https://github.com/nikhil-sarin/redback/blob/master/examples/analyse_spectrums.ipynb>`_ - Spectral analysis
